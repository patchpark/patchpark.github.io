# VBA 세팅
파일 -> 옵션 -> 보안센터 -> 보안센터설정

매크로 설정 -> 모든 매크로 포함, 개발자 매크로 설정 체크
제한된 보기 -> 체크박스 해제

파일 -> 옵션 -> 리본 사용자 지정 -> 개발도구 체크

개발도구 -> Visual Basic(ALT + F11)

보기 -> 직접 실행창


보기 -> 도구 모음 -> 디버그, 편집 추가

도구 -> 옵션 -> 편집기 형식 -> 글꼴(Consolas)
